# SQL Database Architecture & Querying Portfolio

Welcome to my SQL data portfolio. This repository showcases my ability to design relational database schemas, write structured data definition language (DDL) scripts, and write data manipulation queries (DML) across different industry domains using DBeaver.

---

## 🏋️ 1. Fitness Activity Portfolio Tracker
* **Project Type:** Personal Metrics Modeling
* **Database Design:** Created a database to log workouts, track daily steps, monitor caloric expenditure, and evaluate body composition trends over time.
* **Core SQL Concepts:** Table creation, primary keys, relational schema design, data type selection.

## 🏢 2. Corporate Database Infrastructure
* **Project Type:** Business Operations Schema
* **Database Design:** Built a relational model simulating a corporate structure, tracking employee records, departmental hierarchies, and job roles.
* **Core SQL Concepts:** Foreign key constraints, referential integrity, structural normalization.

## 🛡️ 3. Insurance Company Claims Matrix
* **Project Type:** Enterprise Risk Modeling
* **Database Design:** Modeled a structured system for an insurance provider to manage client profiles, policy details, and historical claim processing records.
* **Core SQL Concepts:** Advanced data structuring, table dependencies, logical constraints.
